## Setup
No build tools or dependencies — pure HTML, CSS, and JavaScript.

To deploy on GitHub Pages:
1. Push this repo to GitHub
2. Go to Settings → Pages → Source: `main` branch, `/ (root)`
3. Your site will be live at `https://<username>.github.io/<repo-name>`

**Note:** Replace `YOUR_FORM_ID` in `index.html` with your Formspree form ID for the contact form to work.
Get a free form ID at [formspree.io](https://formspree.io)
